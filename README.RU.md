# TranslateUI

Минималистичное десктопное приложение-переводчик на **Ollama** и **TranslateGemma**.  
Технологии: **.NET 10**, **C# 14**, **Avalonia UI**, **MVVM**. Без платных библиотек.

## Цели
- Перевод текста, файлов и (при наличии модели) изображений.
- Строгая эстетика, адаптивность и поддержка tray.
- Минимальное потребление ресурсов.

## Возможности
- **Перевод текста**: два окна (источник/результат), перевод по кнопке.
- **Перевод файлов**: выбор через диалог/drag&drop, выбор пути результата, ссылка на файл.
- **Перевод изображений**: только при наличии vision-модели.
- **Выбор языков**: список по `translategemma.MD`.
- **Выбор модели**: ползунок с дискретными точками 4B/12B/27B.
- **Загрузка модели**: предупреждение о скачивании, кнопка загрузки.
- **Буфер обмена**: вставка источника, копирование/сохранение результата.
- **Tray**: закрытие скрывает в трей; меню: Открыть/Перевести/Выход.
- **Отладочное логирование**: уровень и путь настраиваются в настройках.
- **Версия**: отображается в настройках и задается в CI при сборке `releases/*.*.*`.

## Быстрый старт (разработка)
- Установить **.NET 10 SDK** и **Ollama**.
- Запустить Ollama (`http://localhost:11434`).
- Скачать модель: `ollama pull translategemma:4b`.
- Сборка: `dotnet build`
- Тесты: `dotnet test TranslateUI.Tests/TranslateUI.Tests.csproj`
