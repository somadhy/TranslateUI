# UI/UX

## 1. Главный экран
**Основной layout:**
- Верхняя панель: выбор языков, модельный ползунок, кнопка “Перевод”.
- Центральная область: два текстовых блока (Source / Result).
- Нижняя строка: статус (модель, прогресс, ошибки), кнопки “Сохранить”, “Открыть результат”.

## 2. Режимы
- **Текст**: два текстовых поля.
- **Файл**: панель выбора файла + путь результата.
- **Изображение**: зона перетаскивания или кнопка выбора (только при vision).

## 3. Ползунок модели
- Дискретные положения 4B/12B/27B.
- Подписи “Скорость” (слева) и “Точность” (справа).
- Визуальный индикатор, если модель отсутствует (иконка/цвет).
- Кнопка “Скачать модель” рядом с ползунком.

## 4. Адаптивность
- Перераспределение пространства между текстовыми окнами.
- Скролл текста без “дребезга”.
- Минимальные размеры окна с предупреждением.

## 5. Tray
- Свертывание в tray без завершения приложения.
- Контекстное меню: “Открыть”, “Перевести”, “Выход”.

## 6. Строгая эстетика
- Сдержанная палитра (серый/графит).
- Минимум декоративных элементов.
- Четкая иерархия текста.
