# План реализации

## Фаза 1: Каркас проекта
- Создать проект Avalonia (.NET 10, MVVM).
- Настроить DI и базовые сервисы.
- Добавить локализацию RU/EN.

## Фаза 2: Перевод текста
- Реализовать `OllamaClient` и `PromptBuilder`.
- Создать ViewModel с двумя текстовыми полями.
- Добавить кнопку “Перевод” и отображение ошибок.

## Фаза 3: Перевод файлов
- Ввести `IFileHandler` и обработчики TXT/MD.
- Добавить UI для выбора файла и пути результата.
- Реализовать сохранение результата.

## Фаза 4: Поддержка расширенных форматов
- Подключить PDF/DOCX/ODT обработчики.
- Добавить обработку изображений при vision-модели.

## Фаза 5: Настройки и модель
- Реализовать ползунок моделей 4B/12B/27B.
- Проверка наличия модели и загрузка через `/api/pull`.
- Настройки по умолчанию и сохранение.

## Фаза 6: Тестирование и полировка
- Покрыть ключевую логику тестами.
- Проверить UX, адаптивность, tray.
- Оптимизировать производительность и ошибки.
- Обеспечить отображение версии в настройках и сборку версии в CI для `releases/*.*.*`.
