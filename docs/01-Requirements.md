# Требования

## 1. Область и ограничения
- Desktop приложение-переводчик текста с доступом к файловой системе и API Ollama.
- Эстетика строгая, но минималистичная.
- Низкая нагрузка по RAM/CPU/Disk.
- Только бесплатные библиотеки.

## 2. Функциональные требования
### 2.1 Перевод текста
- Два окна: исходный текст и результат.
- Перевод запускается только кнопкой **"Перевод"**, без авто-перевода при наборе.
- После перевода предлагать сохранить результат в файл.
- Полноценная работа с буфером обмена.

### 2.2 Перевод файлов
- Выбор файла через диалог или drag&drop.
- Выбор пути результата.
- Показ ссылки/кнопки “Открыть результат”.

### 2.3 Перевод изображений
- Функция доступна только если выбранная модель поддерживает изображения.
- Поддерживаемые форматы: PNG, JPEG, TIFF.

### 2.4 Выбор языка
- Языки источника и назначения выбираются из списка `translategemma.MD`.
- Сохранять последние выбранные языки как значения по умолчанию.

### 2.5 Выбор модели
- Ползунок с дискретными позициями: `translategemma:4b`, `translategemma:12b`, `translategemma:27b`.
- Подписи ползунка: слева “Скорость”, справа “Точность”.
- Если модель отсутствует локально — визуальное предупреждение.
- Предлагать кнопку “Скачать” или авто-скачивать при переводе.

### 2.6 Настройки
- Язык интерфейса (по умолчанию English).
- Язык источника/цели по умолчанию.
- Адрес Ollama сервиса.
- Модель по умолчанию.
- Уровень отладочного логирования (Debug/Info/Warn/Error/Off).
- Отображение версии приложения.

## 3. Поддерживаемые форматы
PDF, TXT, MD, DOCX, ODT, PNG, JPEG, TIFF.  
Обработка должна быть расширяемой (паттерн/стратегия для добавления новых форматов).

## 4. Нефункциональные требования
- Строгий минимализм и адаптивный UI.
- Окно можно изменять по размеру, сворачивать в tray.
- Безопасность при работе с файлами и сетью.
- Высокая читаемость кода, SOLID.

## 5. Критерии приемки
- Перевод текста и файлов выполняется корректно и по кнопке.
- Список языков соответствует `translategemma.MD`.
- Ползунок моделей работает дискретно и отражает наличие модели.
- Настройки сохраняются и применяются при запуске.
- Доступна локализация RU/EN.
